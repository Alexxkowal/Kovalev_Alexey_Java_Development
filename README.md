# Калькулятор отпускных (Тестовое задание в NeoFlex)

Микросервис для расчета компенсации за отпуск.

### Функционал:

1. Расчет суммы отпускных по средней зарплате и количеству дней отпуска.
2. Расчет с учетом производственного календаря (при передаче конкретных дат). В этом случае оплачиваются только рабочие
   дни, а праздничные дни из файла `holidays.txt` и выходные исключаются из расчета.

### Стек:

* Java 17
* Spring Boot
* Maven
* JUnit 5 (для тестов)
